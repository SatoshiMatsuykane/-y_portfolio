Profile

現在は、マーケティングスキルを高めるため、クラウドワークスにてライティング案件に挑戦しています。

  • [Category]

  • [Title]

    STCジュニアテニスアカデミー

  • [Skill]

    HTML / CSS / Javascript

  • [対象者]

    テニスを始めた学生(主に小中学生)

  • [作成した経緯]

    私自身、趣味でテニスをやるのですが、そこで学生が練習をしている所をよく見かけます。
    楽しくやっている子達もいるのですが、逆にすごく怒っていたり、やる気をなくしている子を見る場面をある程度の頻度でみかけます。
    その子たちに共通して見られるのは、自分が思った通りにボールを飛ばせていない状態でした。理由としては、顧問の先生や周りの人たちが テニスが未経験で、技術的指導ができる人がおらず、いつまでも問題が解決できていないからです。
    そんなテニスを始めたての学生に向けて、興味を持ってもらえるように架空のテニススクールのWEBサイトを作りました。

  • [意識したこと]

    ・対象が学生なため、文字フォントは見やすく、シンプルさを特に意識しました。
    ・初級者にガツガツしたイメージを与えたくなかったので、青色をメインカラーにして誠実さを意識
    ・ユーザーがわかりやすいように、aタグには擬似要素をいれました
    ・１つ１つのセクションを分かりやすくするため、余白と背景色を意識しました
    ・スクロールアニメーションをいれ、変化をつけました

  • [苦戦したところ]

    ・トップ画像をスライダーで適用させる部分
    ・ハンバーガーメニューの実装
